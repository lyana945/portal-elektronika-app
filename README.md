# Portal Elektronika App

Aplikasi web berbasis Laravel untuk menyelesaikan permasalahan di lingkungan kampus menggunakan pendekatan MVC.

## Fitur Utama
* Otentikasi Pengguna (Login/Register Multi-role 2 Level)
* 4 Modul CRUD Berelasi (Eloquent ORM)
* Proteksi Rute dengan Middleware
* Validasi Input Sisi Server (Form Request)
* Tampilan Responsif (Blade Templating Engine & CSS Framework)
